# genieacs-box-docker
GenieACS in box

O projeto comtempla a criação de um ambiente em docker, utilizando dockerfile e scripts bash, com a seguinte estrutura:
- NodeJS
- Ruby
- MongoDB
- GenieACS
- GenieACS GUI
